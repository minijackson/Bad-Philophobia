boolean BeamerCommand.execute ( Player  player) throws NoArgumentException,IllegalArgumentException,UnauthorizedException

Teleport to the remembered room or remember a room.

It teleports to the remembered room if the command parameter was "teleport", and it remembers the room if the command parameter was "charge"

Parameters
playerThe player that called this command
Exceptions
NoArgumentExceptionWhen the user typed the command without parameter
IllegalArgumentExceptionWhen the user typed a parameter other than "teleport" or "charge"
UnauthorizedExceptionWhen the user tries to teleport to null
Returns
False because it is not the quit command
