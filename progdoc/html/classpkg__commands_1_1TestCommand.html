boolean pkg_commands.TestCommand.execute ( Player  player) throws NoArgumentException,pkg_exceptions.IllegalArgumentException

Test series of Commands stored in a .test file and store the messages in the message field.

Each command in the test file must be separated by an end-of-line character

Parameters
playerThe player that called this command
Exceptions
NoArgumentExceptionWhen the user typed the command without parameter
IllegalArgumentExceptionWhen the user typed a name other than any of the .test files in the directory
Returns
False because it is not the quit command
